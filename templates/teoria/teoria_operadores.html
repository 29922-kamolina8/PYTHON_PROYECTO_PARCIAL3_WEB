2. Operadores de Comparación

Comparan dos valores y devuelven un Booleano (True o False).

a = 10
b = 20

print(a == b) # False (Igualdad)
print(a != b) # True (Diferencia)
print(a > b) # False (Mayor que)
print(a <= 10) # True (Menor o igual)

3. Operadores Lógicos

Permiten combinar múltiples condiciones.

  • and (Y) True si ambos son True
  • or (O) True si al menos uno es True
  • not (NO) Invierte el valor (True -> False)
