Variables

Una variable es como una "caja" donde guardamos un dato. En Python, no necesitas declarar el tipo (es de tipado dinámico).

# Creación de variables
nombre = "Juan"   # Es un String
edad = 25         # Es un Entero
precio = 19.99    # Es un Float

# Podemos cambiar el tipo sobre la marcha
edad = "Veinticinco" # Ahora es String (No recomendado pero posible)

Reglas de Nombrado (Snake Case)

En Python, por convención usamos snake_case (minúsculas separadas por guiones bajos).

nombre_usuario
nombreUsuario

Constantes

Python no tiene constantes "reales" (que no se puedan cambiar), pero por convención, si escribes una variable en MAYÚSCULAS, estás gritando: "¡NO CAMBIES ESTO!".

PI = 3.1416
CONEXION_DB = "localhost:3306"
