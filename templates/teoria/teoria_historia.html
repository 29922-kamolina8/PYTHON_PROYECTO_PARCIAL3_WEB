El Origen (1989)

Python fue creado por Guido van Rossum en los Países Bajos durante la Navidad de 1989. Guido buscaba un proyecto de programación para mantenerse ocupado durante las vacaciones.

El nombre "Python" no viene de la serpiente, sino del grupo de comedia británico Monty Python's Flying Circus, del cual Guido era fanático.

Python 2 vs Python 3

Un momento crítico en la historia fue el salto de la versión 2 a la 3.

  • Python 2.0 (2000): Introdujo recolección de basura y soporte unicode básico. Dejó de tener soporte en 2020.
  • Python 3.0 (2008): Diseñado para rectificar fallas fundamentales. No era retrocompatible (código de Python 2 no funcionaba en Python 3).
  • Actualidad: Hoy usamos Python 3.10+, que es rápido, moderno y robusto.
