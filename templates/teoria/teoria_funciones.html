¿Qué es una función?

Es un bloque de código con un nombre, que puede recibir datos (parámetros) y devolver un resultado.

def saludar(nombre):
    """Esta función saluda a alguien"""
    return f"Hola, {nombre}!"

# Uso de la función
mensaje = saludar("Carlos")
print(mensaje) # Salida: Hola, Carlos!

Componentes Clave

  • def: Palabra reservada para definir la función.
  • return: Finaliza la función y devuelve un valor al código principal.
  • Parámetros: Variables que la función espera recibir entre paréntesis.

Scope (Alcance)

Scope (Alcance): Las variables creadas DENTRO de una función solo existen allí. No puedes usarlas fuera.
