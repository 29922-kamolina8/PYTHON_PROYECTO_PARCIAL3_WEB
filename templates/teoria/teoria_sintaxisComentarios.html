La Indentación

A diferencia de Java o C++ que usan llaves {} para definir bloques de código, Python usa la indentación (sangría). Esto obliga a escribir código ordenado.

✅ Correcto
if 5 > 2:
    print("Cinco es mayor")
❌ Error (IndentationError)
if 5 > 2:
print("Cinco es mayor")
Nota: Se recomienda usar 4 espacios por nivel de indentación.

Comentarios

Los comentarios son ignorados por el intérprete y sirven para documentar tu código.

  • Una línea: Usamos el símbolo almohadilla #.
    # Esto es un comentario
    print("Hola") # Esto también
  • Multilínea (Docstrings): Usamos tres comillas dobles """.
    """
    Este es un comentario
    que ocupa varias líneas.
    """
