Condicionales (if / elif / else)

Permiten ejecutar código solo si se cumple una condición.

edad = 18

if edad < 18:
    print("Eres menor de edad")
elif edad == 18:
    print("Acabas de cumplir la mayoría")
else:
    print("Eres adulto")

Bucle For

Se utiliza para iterar sobre una secuencia (lista, string, rango).

# Imprimir números del 0 al 4
for i in range(5):
    print(i)

Bucle While

Repite el código mientras una condición sea verdadera. ¡Cuidado con los bucles infinitos!

contador = 5
while contador > 0:
    print(f"Cuenta regresiva: {contador}")
    contador -= 1
