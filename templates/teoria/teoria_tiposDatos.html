En Python todo es un objeto, pero existen tipos de datos primitivos esenciales que usarás el 99% del tiempo.

Tipos Primitivos

str
String (Texto)

Cadenas de texto entre comillas simples o dobles.

x = "Hola Mundo"
int
Integer (Entero)

Números enteros sin decimales, positivos o negativos.

edad = 25
float
Float (Flotante)

Números con punto decimal.

precio = 19.99
bool
Boolean (Lógico)

Solo dos valores posibles. Nota la mayúscula.

activo = True

Verificar el tipo

Puedes usar la función type() para saber qué es una variable.

print(type(10))    # <class 'int'>
print(type(3.14)) # <class 'float'>
